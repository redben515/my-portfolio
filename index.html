GamerPocket

Android application

Gamer pocket is an android application that helps gamers to keep track the newest information and release date of their favourite games. The core functionality will be sending out notification about how many days until their favourite game release. The app is using Giant Bomb API to get the game information. It will also allow user to see the latest news, game screenshots, and game box art.

All the game companies nowadays will let their fans know whenever they are going to release a game in order to build up “hype” for their games. Even though the game is not released, they will still have a lot of news and screenshots to keep their audience interested and looking forward to play their games. This application is aim to provide a countdown watch to keep track of how many days until their desired game publish to the market. On the other hand, preordering a game is pretty common right now, this application can give those gamers out there a tool to manage their pre-ordered games.

  • Plaform: Android
  • Programming Language: Java
  • Tool: Android Studio

Fleashelf

A textbook trading platform for students.

Fleashelf is a website that provides a platform which allows Seneca students to trade, buy and sell used books. Once a student registers for an account they will be able to contact fellow students who are buying and selling books. They will also be able to list a book for sale. As a seller a student will upload their used book information as well as their contact info. As a buyer a student will be able to search for books based on title, category, or course code.

  • Platform: Web
  • Programming Language: C#, .Net
  • Tool: Visual Studio
